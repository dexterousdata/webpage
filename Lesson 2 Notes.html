Lesson 2.2 - Variables and Strings

Variable

In Python a variable is an item that refers to another object. Similar to math’s.
a could reference the integer 10 or a could reference the string "abc", and many other possibilities

Assign a Variable

You assign a variable through the " = " sign. a = 10 translates to the symbol "a" is now bonded to the integer value of 10. So at this point "a" and 10 have the same value.

"=" when assigning a variable is different to math’s were you are stating 2 + 3 = (is equal to) 5. Assigning binds a value to a variable so a = a - 1 is one less than a's current value. In math’s this is not possible,

Variable Uses

Variable's are used to add redability to code. Transforming a number or string it something easily recognizable

To note:
2 + 2 and "2" + "2" are very different. The first is integer addition and the second is concatenation of strings

Lesson 2.3 - Procedures

Functions

Functions or 'procedures' are sub programs within the larger program. They act to complete a task for each given input. The top program can call on a sub program at any time and give inputs. The function then provides the output the top program uses.

Making a function is creating the sub program. The internal procedures for the function. Calling the function or using it is 'running' the sub program to provide an out put that is used by the program that called it.

Avoiding Repetition

Functions help programmers avoid repetition by preforming a sub task for each and every given input instead of the programmer rewriting the code for each input.

Return Statement

The return statement tells the sub program what to return to the program that called it. The program that called the function then will use the provided output in its code.
